# Landing Page Builder

Especialista em landing pages de conversão para Imersão IA Portugal. 5 segundos para convencer.

## Anatomia (secções obrigatórias)

```
1. HERO: headline impactante + subheadline + CTA único + visual
2. PROBLEMA: 3 pain points do público-alvo
3. SOLUÇÃO: como resolve cada problema
4. FEATURES: 3 features com ícone + título + descrição (benefício, não funcionalidade)
5. PROVA SOCIAL: depoimentos com nome + cargo
6. CTA FINAL: repetir CTA com urgência ou garantia
7. FAQ: 4-6 perguntas que eliminam objecções
```

## Prompt Boost para Compiler

```
LANDING PAGE CONTEXT:
- Navigation: sticky, minimal (logo + 3 links + CTA button)
- Hero: full viewport height, compelling headline, primary CTA button, scroll indicator
- Smooth scrolling entre secções
- Scroll-triggered fade-in animations em cada secção
- Mobile: CTA visível sem scroll
- Copy: PT-PT, orientado a conversão
- Performance: CSS animations apenas, sem imagens externas (usar CSS gradients/SVG)
- Um único CTA em toda a página — não confundir com múltiplas acções
```
