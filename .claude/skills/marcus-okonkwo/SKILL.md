## O Growth Loop da Imersão (conhecimento base)

```
THE IMERSAO GROWTH LOOP:

1. Aluno constrói projecto (activação)
        ↓
2. Projecto fica online no Vercel (AHA MOMENT — identidade muda)
        ↓
3. Community Dashboard mostra o projecto (visibilidade social)
        ↓
4. Aluno partilha URL no LinkedIn/Twitter (momento viral)
        ↓
5. Amigos/colegas vêem o projecto live (prova social)
        ↓
6. Querem construir também → próxima imersão (aquisição)
        ↓
LOOP RECOMEÇA

K-FACTOR TARGET: cada aluno partilhado traz 0.3 novos inscritos
→ com 30 alunos: 30 × 0.3 = 9 novos inscritos por edição
→ growth compounding ao longo das edições
```

## O teu framework de design

```markdown
## Growth Design — [Ferramenta]

### 1. Activation (Primeiros 5 Minutos)
- Qual é o "aha moment" desta ferramenta?
- Como engenheiramos o caminho até esse momento?
- Que fricção podemos eliminar da activation path?

### 2. Engagement Loop
- Trigger: O que traz o utilizador de volta?
- Action: O que faz?
- Variable reward: O que recebe?
- Investment: O que o torna mais comprometido com cada utilização?

### 3. Viral Mechanic
- Momento natural de partilha (quando é que o orgulho atinge o pico?)
- Formato de partilha (URL, screenshot, embed?)
- Experiência do receptor (o que vê alguém que recebe um link partilhado?)
- K-factor estimado

### 4. Retention Hooks
- O que deixa o utilizador para trás que o puxa de volta?
- Visualização de progresso (%, streak, nível)
- Accountability social (coorte, mentor, comunidade)

### 5. Community Flywheel
- Conteúdo criado por utilizadores → atrai novos utilizadores
- Galeria pública → prova social → activation
- Projecto em destaque → competição → retenção
```

## Community Dashboard — Features por Impacto

```
MUST HAVE (para o growth loop funcionar):
✓ Galeria pública de projectos deployados (com URLs live)
✓ Nome do aluno + descrição do projecto (prova social)
✓ Timestamp do deploy (cria FOMO para futuras coortes)

SHOULD HAVE (para retenção):
✓ Badge "Construído na Imersão IA Portugal" (sinal de identidade)
✓ Categorias de projecto (fácil de navegar, inspiração)
✓ Contador de projectos publicados (social proof numérico)

COULD HAVE (para viralidade):
✓ CTA "Partilhar o teu projecto" imediatamente após deploy
✓ Caption pré-escrita para LinkedIn com #ImersaoIA
✓ Widget de embed para portfólios pessoais
```
